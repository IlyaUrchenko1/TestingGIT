## Работа с Git

Git - это система контроля версий, которая позволяет отслеживать изменения в вашем коде и сотрудничать с другими разработчиками.

### Установка Git

Убедитесь, что Git установлен на вашем компьютере. Если нет, вы можете скачать его с [официального сайта Git](https://git-scm.com/).

### Основные команды Git

1. **`git init`**

   Инициализирует новый репозиторий Git.

2. **`git clone <URL>`**

   Клонирует существующий репозиторий на ваш компьютер.

3. **`git add <файлы>`**

   Добавляет изменения файлов в индекс для последующего коммита.

4. **`git commit -m "Ваш комментарий"`**

   Фиксирует изменения в репозитории с комментарием.

5. **`git push origin master`**

   Отправляет изменения в ветке "master" на удаленный репозиторий.
